Automated Student Attendance System

Overview

The Automated Student Attendance System is a web-based application that simplifies attendance management using QR code technology. It enables faculty members to generate secure QR codes for each class session, while students can scan the QR code to mark their attendance quickly and accurately.

Features

- Secure login for Admin, Faculty, and Students
- QR Code-based attendance system
- Attendance validation to prevent duplicate entries
- Student attendance dashboard
- Faculty dashboard for attendance management
- Attendance analytics and reports
- Low attendance alerts
- Session management with expiry time
- Unit testing for application reliability

Installation

1. Clone the repository

git clone https://github.com/Prayukta887/Automated-Student-Attendance-System.git

2. Open the project folder

cd Automated-Student-Attendance-System

3. Create a virtual environment

python -m venv venv

4. Activate it

Windows:

venv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

6. Run the application

python app.py
